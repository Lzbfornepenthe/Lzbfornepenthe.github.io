## 🎉 我的个人博客，今日正式上线！

你好，世界！这是我的 Hexo 博客的第一篇文章。经过一番配置，这个由 **Hexo** 驱动、托管在 **GitHub Pages** 上的静态博客终于成功运行了。

### 为什么选择这个技术栈？

- **完全免费**：GitHub Pages 提供了稳定的静态站点托管服务。
- **极简高效**：Hexo 生成静态页面的速度非常快。
- **专注写作**：用熟悉的 Markdown 语法来撰写内容，无需关心排版。
- **高度自由**：可以随心所欲地更换主题、定制样式。

### 搭建过程回顾
主要步骤可以概括为以下几点：
1.  **环境准备**：安装 Node.js 和 Git。
2.  **连接 GitHub**：配置 SSH 密钥实现免密推送。
3.  **创建仓库**：建立 `用户名.github.io` 这个特殊命名的仓库。
4.  **本地安装**：使用 npm 安装 Hexo 并初始化博客。
5.  **部署上线**：修改 `_config.yml` 配置文件，一键部署到 GitHub。

### 接下来的计划
1.  **探索主题**：默认的 `landscape` 主题不错，但打算试试更流行的 `NexT` 或 `Butterfly` 主题。
2.  **添加插件**：考虑安装搜索、评论（如 Gitalk、Waline）、文章统计等插件。
3.  **绑定域名**：也许会给博客换一个更个性的专属域名。

> 千里之行，始于足下。这个博客将记录我的学习、思考与成长。
